HistogramVAEWrapper#

class capsa.HistogramVAEWrapper(*args, **kwargs)[source]#

Combines the functionalities of both HistogramWrapper and VAEWrapper. This is done by converting a given base_model into Variational AutoEncoder architecture, with latent dimension defined by the given parameter latent_dim.

VAEs are typically used to learn a robust, low-dimensional representation of the latent space. They can be used as a method of estimating epistemic uncertainty by using the reconstruction loss MSE(x, x_hat) - in cases of out-of-distribution data, samples that are hard to learn, or underrepresented samples, we expect that the VAE will have high reconstruction loss, since the mapping to the latent space will be less accurate. Conversely, when the model is very familiar with the features being fed in, or the data is in distribution, we expect the latent space mapping to be robust and the reconstruction loss to be low.

The logic for one training step.
