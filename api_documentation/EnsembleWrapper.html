EnsembleWrapper#

Note

For more examples of using the EnsembleWrapper check out capsa/tests/test_ensemble.py.

class capsa.EnsembleWrapper(*args, **kwargs)[source]#

Uses an ensemble of N models (each one is randomly initialized) to accurately estimate epistemic uncertainty Lakshminarayanan et al. (2017).

This approach presents the gold-standard of estimating epistemic uncertainty. However, it comes with significant computational costs.

Example of usage:
>>> # initialize a keras model
>>> user_model = Unet()
>>> # wrap the model to transform it into a risk-aware variant
>>> model = EnsembleWrapper(user_model, metric_wrapper=MVEWrapper, num_members=3)
>>> # compile and fit as a regular tf.keras model
>>> model.compile(...)
>>> model.fit(...)
