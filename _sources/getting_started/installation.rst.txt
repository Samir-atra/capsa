Installation
============


There are several different ways to install capsa into your workspace.

.. tabs::

   .. tab:: pip

    Install capsa with Python's pip package manager. This will install the latest published release available.

      .. code-block:: bash

         pip install capsa

   .. tab:: conda

    Install capsa with conda.

      .. code-block:: bash

         conda install -c themisai capsa

   .. tab:: source

    Install capsa from Github.

      .. code-block:: bash

         pip install git+https://github.com/themisai/capsa.git
